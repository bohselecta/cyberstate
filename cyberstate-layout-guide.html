▸ MODULE HIERARCHY

  • HEADER MODULE - 80px fixed height, spans full width
    • Logo (200px) | Search Bar (400px) | Filter Chips (flex) | System Status (240px)
  • LEFT SIDEBAR - 320px fixed width, full height minus header
    • Listing Queue - Virtual scrolling list of property cards
    • Each card: 120px height with thumbnail + info grid
  • MAIN CONTENT - Fluid width between sidebars
    • Map View (60% default) - Expandable to 100%
    • Property Viewer (40% default) - Expandable to 70%
  • RIGHT SIDEBAR - 320px fixed width, vertically scrollable
    • Stack of collapsible data modules
    • Each module: 180-280px collapsed height
    • 7 modules total (can add more)

▸ PROPERTY VIEWER BREAKDOWN

  • Photo Carousel - 65% of viewer height
    • Main image: 16:9 aspect ratio with glow border
    • Thumbnail strip: 80px height at bottom
    • Navigation: Arrow overlays + keyboard support
  • Property Header - 120px fixed height
    • Price (H1 bold glow) | Address (H3)
    • Quick stats: Beds | Baths | Sqft | Year Built
    • Action buttons: Schedule Tour | Save | Share
  • Property Tabs - Remaining height
    • Overview: Description + Property Details grid
    • Price History: Line chart showing price changes
    • Comps: Grid of comparable property cards

▸ COLOR CODING

Header region - always visible
Interactive regions - click to view details
24-column grid lines (toggle on/off)

▸ MOBILE ADAPTATION (< 768px)

  • Single column stack - all modules become full width
  • Map stays at top: 40vh height, touch zoom/pan enabled
  • Selected property title: Sticky header below map
  • All modules collapse by default: Tap header to expand
  • Module order: Photo Carousel → Details → Analytics
  • Floating action buttons: Filter drawer + Listings drawer
